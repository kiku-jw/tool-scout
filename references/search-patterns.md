# Search Patterns

Use current-year queries and prefer primary sources.

## Source priority

1. official documentation
2. official GitHub repository
3. official pricing page
4. reputable ecosystem comparison only as supporting context

## Query patterns

- `best [task] tool [current year]`
- `[task] MCP server`
- `[task] API official`
- `[task] open source GitHub`
- `[task] pricing official`

## When to browse

Always browse for this skill. Tool recommendations are temporally unstable.
